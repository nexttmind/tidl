 Basic embed
The minimum viable integration. Initialize the SDK with an embedCode , wire the four lifecycle
callbacks (onReady , onStepChange , onComplete , onError ), and the wizard runs end-to-end.
Every event fired by the iframe is mirrored in the live event log on the right.
The integration code
Drop this into any HTML page; replace YOUR_EMBED_CODE .
<!-- 1. The container the SDK will render the iframe into -->
<div id="prx-intake"></div>
<!-- 2. Load the SDK -->
<script src="https://demo.prescribe-rx.com/embed/sdk.js"></script>
<!-- 3. Initialize -->
<script>
  PrescribeRx.init('prx-intake', {
    embedCode: 'YOUR_EMBED_CODE',
    onReady:      (data) => console.log('Ready', data),
    onStepChange: (data) => console.log('Step ' + data.step + '/' + data.total
    onComplete:   (data) => console.log('Done!', data),
    onError:      (data) => console.error('Error:', data.message),
  });
</script>
