 Preselected package
Drop the user directly into a specific package + plan instead of making them pick one in-
wizard. Useful when your storefront already presented packages and the user clicked "Buy
now". The wizard's package-picker step is skipped automatically because a package is already
selected.
The integration pattern
Two valid shapes — pass at init time, or set at runtime.
// Option A — at init time (loads the wizard already inside the package)
PrescribeRx.init('prx-intake', {
  embedCode: 'YOUR_EMBED_CODE',
  package:   'PKG-00123',         // human-friendly
});
// Option B — runtime, after the user clicks something on YOUR page
PrescribeRx.selectPackage('PKG-00123');
PrescribeRx.selectPlan('019d4f...uuid');   // optional — pin the subscription 
// Option C — multiple packages (rarely needed; bundles)
PrescribeRx.selectPackages(['PKG-001', 'PKG-002']);
Try it — pick a package

plan picker is auto-populated from /api/v1/packages/{id} after you enter a valid package number
— no fake placeholders.
